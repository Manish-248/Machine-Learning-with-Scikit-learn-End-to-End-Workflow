# 🚀 Machine Learning Fundamentals with Scikit-learn

### End-to-End Workflow Implementation

---

## 📌 Project Overview

This project presents a **complete, end-to-end Machine Learning workflow** implemented using **Scikit-learn**. It demonstrates the full lifecycle of a typical ML pipeline — from data preprocessing to model evaluation and dimensionality reduction.

The goal of this project is to showcase **practical, production-oriented machine learning skills**, with a strong focus on clean code, reproducibility, and best practices.

---

## 🎯 Objectives

* Understand and implement the **Scikit-learn Estimator API**
* Build a **complete ML pipeline**
* Apply **data preprocessing techniques**
* Train and evaluate **multiple machine learning models**
* Implement **regularization techniques**
* Perform **dimensionality reduction using PCA**
* Follow **industry-standard coding practices**

---

## 🧠 Key Concepts Covered

### 🔹 1. Data Handling & Exploration

* Loading datasets (Iris dataset)
* Understanding feature matrix (X) and target vector (y)
* Dataset inspection and structure

---

### 🔹 2. Train-Test Splitting

* Splitting data into training and testing sets
* Preventing data leakage
* Ensuring model generalization

---

### 🔹 3. Scikit-learn Estimator API

Core concept of the project:

* `fit()` → Model training
* `predict()` → Making predictions
* `transform()` → Data preprocessing

---

### 🔹 4. Data Preprocessing

* Feature scaling using `StandardScaler`
* Importance of normalization in ML models
* Improving model performance through preprocessing

---

### 🔹 5. Machine Learning Pipeline

* Combining preprocessing and model steps
* Using `Pipeline` for cleaner workflows
* Industry-standard implementation

---

### 🔹 6. Linear Models

#### ✔ Linear Regression

* Continuous value prediction
* Model evaluation using Mean Squared Error (MSE)

#### ✔ Logistic Regression

* Binary classification
* Decision boundary understanding

---

### 🔹 7. Regularization Techniques

#### ✔ Ridge Regression (L2)

* Reduces model complexity
* Prevents overfitting

#### ✔ Lasso Regression (L1)

* Performs feature selection
* Produces sparse models

---

### 🔹 8. Model Evaluation

* Accuracy score (classification)
* Mean Squared Error (regression)
* Performance comparison

---

### 🔹 9. Dimensionality Reduction (PCA)

* Reducing high-dimensional data
* Preserving maximum variance
* Visualizing data in 2D
* Explained variance interpretation
